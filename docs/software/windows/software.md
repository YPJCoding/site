# Windows 软件清单

这篇文档用来记录 Windows 系统中值得安装和长期使用的软件，方便重装系统、换电脑或者整理工作环境时快速参考。

## UniGetUI

UniGetUI 是一款 Windows 图形化软件包管理工具，可以把常见的包管理器能力集中到一个界面里使用，例如 WinGet、Scoop、Chocolatey 等。

它适合用来统一搜索、安装、更新和卸载软件，减少手动打开官网下载安装包的次数，也方便批量维护电脑上的常用工具。

- 类型：Windows 软件管理 / 包管理器图形界面
- 适合场景：新电脑初始化、批量安装常用软件、统一检查软件更新
- 官方链接：[UniGetUI 官方仓库](https://github.com/marticliment/UniGetUI)
