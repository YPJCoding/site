# 插件清单

浏览器是我日常工作和上网最核心的入口之一。相比频繁更换新工具，我更倾向于保留一套长期稳定、真正高频使用的扩展组合：能解决明确问题、不会明显拖慢浏览器、并且能够融入我的使用习惯。

这篇文章记录一下我目前常用的一批浏览器插件。它们不一定是“最全”的选择，但基本覆盖了我在 **网页阅读、内容过滤、脚本增强、书签同步、下载接管、开发辅助** 等方面的主要需求。

> 我的插件选择原则：
>
> 1. **足够稳定**：优先选择长期维护、口碑稳定的扩展。
> 2. **高频实用**：能显著减少重复操作。
> 3. **轻量明确**：功能聚焦，不追求花哨堆料。
> 4. **可组合**：插件之间互不冲突，能够形成一套顺手的工作流。

## 插件总览

| 名称                     | 主要作用                  | 我为什么保留它              |
|------------------------|-----------------------|----------------------|
| 沉浸式翻译                  | 网页/PDF 双语翻译           | 阅读外文内容非常高频，能明显提高效率   |
| uBlock Origin          | 广告与无关元素过滤             | 让网页更干净、更快、更专注        |
| Markdown Viewer        | 直接在浏览器里查看 Markdown    | 打开 `.md` 文件或文档链接时更自然 |
| Tampermonkey（篡改猴）      | 用户脚本管理                | 很多页面增强都依赖它           |
| Dark Reader            | 为网站提供暗色模式             | 夜间阅读和长时间浏览更舒服        |
| LinuxDo Scripts        | Linux.do 社区增强脚本       | 优化特定社区的浏览体验          |
| floccus bookmarks sync | 跨浏览器同步书签              | 在不同设备和浏览器之间保持一致      |
| Sunsetvault            | 内容保存/收藏类辅助工具          | 方便沉淀网页内容与个人资料库       |
| Aria2 Explorer         | 调用 Aria2 接管下载         | 下载更可控，适合大文件和批量任务     |
| Infinity 新标签页（Pro）     | 新标签页美化与信息聚合           | 让新标签页更实用，而不只是空白页     |
| Chromium Web Store     | 为 Chromium 系浏览器补充商店支持 | 在部分环境中安装/管理扩展更方便     |

## 1. [沉浸式翻译](https://chrome.google.com/webstore/detail/bpoadfkcbjbfhfodiogcnhhhpibjhbnh)

- **类型**：翻译 / 阅读增强
- **作用**：支持网页翻译、双语对照、PDF 翻译等，适合阅读英文技术文档、博客和论文。
- **适合场景**：
  - 阅读国外技术博客
  - 查看英文产品文档
  - 翻译 PDF 资料

## 2. [uBlock Origin](https://chrome.google.com/webstore/detail/cjpalhdlnbpafiamejdnhcphjbkeiagm)

- **类型**：广告过滤 / 内容拦截
- **作用**：拦截广告、跟踪脚本、弹窗和部分无关页面元素。
- **适合场景**：
  - 日常浏览资讯站点
  - 屏蔽弹窗和追踪脚本
  - 保持页面简洁

## 3. [Markdown Viewer](https://chrome.google.com/webstore/detail/ckkdlimhmcjmikdlpkmbgfkaikojcbjk)

- **类型**：文档阅读
- **作用**：在浏览器中直接以更友好的方式查看 Markdown 文件内容。
- **适合场景**：
  - 查看项目 README
  - 快速浏览在线 Markdown 文档
  - 临时预览笔记内容

## 4. [Tampermonkey（篡改猴）](https://chrome.google.com/webstore/detail/dhdgffkkebhmkfjojejmpbldmpobfkfo)

- **类型**：脚本扩展 / 页面增强
- **作用**：运行用户脚本，对网页行为、样式和交互进行自定义增强。
- **适合场景**：
  - 页面样式微调
  - 自动化重复操作
  - 社区站点增强

## 5. [Dark Reader](https://chrome.google.com/webstore/detail/eimadpbcbfnmbkopoojfekhnkhdbieeh)

- **类型**：护眼 / 显示增强
- **作用**：为大多数网站生成暗色模式，减少强光刺激。
- **适合场景**：
  - 夜间阅读
  - 长时间查资料
  - 降低亮色页面造成的疲劳感

## 6. [LinuxDo Scripts](https://chrome.google.com/webstore/detail/fbgblmjbeebanackldpbmpacppflgmlj)

- **类型**：站点增强 / 用户脚本配套
- **作用**：为 Linux.do 社区提供额外增强功能，优化浏览与交互体验。
- **适合场景**：
  - 论坛浏览增强
  - 提高社区站点可读性
  - 优化交互细节

## 7. [floccus bookmarks sync](https://chrome.google.com/webstore/detail/fnaicdffflnofjppbagibeoednhnbjhg)

- **类型**：书签同步
- **作用**：在不同浏览器、不同设备之间同步书签数据。
- **适合场景**：
  - 多浏览器共用书签
  - 多设备同步收藏夹
  - 统一个人资料入口

## 8. [Sunsetvault](https://chrome.google.com/webstore/detail/leedpmmbobncoipdjocgcbkmogekjkga)

- **类型**：密码管理 / 账号安全
- **作用**：一个基于开源密码管理器构建的定制版本扩展，主要用于连接特定服务端，进行账号、密码等敏感信息的管理。
- **适合场景**：
  - 懂得都懂

## 9. [Aria2 Explorer](https://chrome.google.com/webstore/detail/mpkodccbngfoacfalldjimigbofkhgjn)

- **类型**：下载增强
- **作用**：将浏览器下载任务交给 Aria2 处理，支持更灵活的下载控制。
- **适合场景**：
  - 下载镜像、压缩包、大型资源文件
  - 批量下载
  - 配合本地下载服务使用

## 10. [Infinity 新标签页（Pro）](https://chrome.google.com/webstore/detail/nnnkddnnlpamobajfibfdgfnbcnkgngh)

- **类型**：界面美化 / 起始页增强
- **作用**：自定义新标签页布局，可放置快捷入口、待办、搜索和常用信息模块。
- **适合场景**：
  - 建立个人浏览器起始页
  - 聚合常用链接
  - 提升新标签页的信息密度

## 11. [Chromium Web Store](https://raw.githubusercontent.com/NeverDecaf/chromium-web-store/master/updates.xml)

- **类型**：扩展商店支持
- **作用**：帮助部分 Chromium 系浏览器更方便地访问、安装或兼容 Chrome Web Store 扩展。
- **适合场景**：
  - 在 Chromium 系浏览器中安装扩展
  - 兼容 Chrome Web Store 生态
  - 管理扩展来源
