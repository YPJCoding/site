## 阅读时先记住这 4 句话

1. 中国信保不是普通商业保险公司，它是我国唯一的政策性出口信用保险机构。
2. 出口信用保险的核心，不是保“货坏没坏”，而是保“钱能不能收回来”。
3. 业务主线可以先粗暴理解成：`接单 -> 投保 -> 限额 -> 出运 -> 收汇 -> 出险时报损/索赔 -> 理赔 -> 追偿`。
4. `信步天下` 更像移动端服务与信息触达工具，`信保通` 更像正式办理线上业务的营业厅。

## 这套资料怎么标注“确定”和“推断”

- `可确认事实`：能在中国信保官网、年报、产品说明、官方操作指南里找到依据。
- `高概率推断`：公开资料没有完全写死，但结合官方信息和常见业务系统设计逻辑，基本可以这样理解。

如果以后你拿到了内部原型、接口字段、菜单截图，就优先回填那些“高概率推断”的地方。
