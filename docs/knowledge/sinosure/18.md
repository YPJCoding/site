# 信保业务高频词汇

| 词语      | 人话解释                    | 系统里常出现在哪   |
|---------|-------------------------|------------|
| 信用限额    | 中国信保愿意为某个买方承担风险的额度上限    | 限额申请、审批、出运 |
| 限额审批    | 中国信保判断某买方能不能批额度、批多少     | 限额流程       |
| 可能损失通知书 | 企业发现风险后先提交的报损材料         | 可损通知、案件    |
| 报损      | 告诉中国信保“这笔交易出风险了”        | 风险案件       |
| 可损      | 业务口语，表示这笔交易已进入可能造成损失的状态 | 风险预警、报损    |
| 索赔      | 企业正式申请赔款                | 索赔页面       |
| 核赔      | 中国信保审核责任和金额             | 理赔结果       |
| 减损      | 风险发生后尽量减少损失扩大           | 理赔、追偿      |
| 追偿      | 赔后向责任方追回欠款              | 追偿页面       |
| 资信调查    | 查询买方或银行的信用信息            | 买方管理、前置风控  |
| 国别风险    | 某个国家层面的政治、汇兑、社会等风险      | 国别页面、风控    |
| 保单融资    | 用投保后的交易去支持融资            | 融资模块       |

## 最容易混的 3 组词

### 报损 / 索赔 / 理赔

- 报损：先报出险
- 索赔：正式申请赔款
- 理赔：中国信保审核处理

### 逾期 / 可损

- 逾期：时间状态
- 可损：风险状态

### 保单限额 / 信用限额

- 保单限额：更偏整张保单层面
- 信用限额：更偏具体买方层面
