# 出口贸易基础词汇

| 词语    | 人话解释               | 系统里常出现在哪   |
|-------|--------------------|------------|
| 出口商   | 中国这边卖货或提供服务的企业     | 客户、被保险人    |
| 买方    | 海外采购方、付款义务相关方      | 买方管理、限额、案件 |
| 开证行   | 信用证场景里负责开证和付款责任的银行 | 信用证交易、理赔   |
| 信用证   | 银行参与的一种付款安排        | 支付方式字段     |
| OA 赊账 | 先发货后付款的一类交易方式      | 支付方式、限额    |
| D/P   | 付款交单，一种国际结算方式      | 支付方式       |
| 账期    | 从出运到应付款之间的信用期限     | 出运、收汇      |
| 出运    | 货物发出或服务已交付         | 出运申报       |
| 收汇    | 把海外货款真正收回来         | 收汇登记       |
| 应收账款  | 企业本该收到但还没收回的钱      | 风险管理、融资、理赔 |
| 逾期    | 到期还没收款             | 逾期跟踪、风险预警  |
| 拒收    | 买方拒绝接受货物或服务        | 可损、索赔      |

## 你先重点记哪几个

- 买方
- 开证行
- 账期
- 出运
- 收汇
- 应收账款
