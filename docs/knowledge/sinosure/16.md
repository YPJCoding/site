# 保险基础词汇

## 用法说明

这一篇只做你最常见基础保险词的“人话翻译”，不追求学术定义。

| 词语     | 人话解释                  | 系统里常出现在哪   |
|--------|-----------------------|------------|
| 保险人    | 提供保险保障的一方，在这里通常就是中国信保 | 保单、条款、理赔文书 |
| 被保险人   | 享受保险保障的一方，很多时候是中国出口企业 | 保单、索赔、赔款   |
| 保单     | 企业和中国信保之间的保险合同        | 保单详情、出运、索赔 |
| 保险责任   | 哪些损失属于“该赔”的范围         | 条款说明、核赔结果  |
| 除外责任   | 哪些情况“不赔”或可能少赔         | 条款说明、核赔争议  |
| 保险期间   | 这张保单在什么时间段内有效         | 保单详情       |
| 赔偿比例   | 核赔时按什么比例赔             | 保单明细、理赔    |
| 最高赔偿限额 | 整体赔付的上限控制             | 保单明细       |
| 赔付基数   | 赔款计算时所依据的金额基础         | 核赔         |
| 理赔     | 中国信保审核后处理赔付           | 案件流程       |
| 追偿     | 赔后继续去追回欠款             | 追偿页面       |
| 代位追偿权  | 赔付后，保险人依法接手部分追偿权利     | 理赔、法务资料    |

## 你先重点记哪几个

最先要熟的是：

- 保单
- 被保险人
- 保险责任
- 赔偿比例
- 理赔
- 追偿
