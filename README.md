- 站点配置拆分清晰，便于长期维护
- 首页组件层与正文组件层已建立
- 适合作为中文文档站模板起点

## 技术栈

- VitePress 2.0.0-alpha.17

## 本地开发

安装依赖：

```bash
pnpm install
````

启动开发环境：

```bash
pnpm run docs:dev
```

构建站点：

```bash
pnpm run docs:build
```

预览构建结果：

```bash
pnpm run docs:preview
```
